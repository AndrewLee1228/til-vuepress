## 개발인생
8살 때 아버지를 따라 코딩을 시작하고, 15년째 코딩을 하고있는 이재민입니다.

컴퓨터에 관한것이라면 다 좋아해서 안해본 것 없이 살다가 최근 들어서 자바스크립트에 정착해있습니다.
